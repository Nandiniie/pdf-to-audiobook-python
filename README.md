# PDF to Audiobook Converter

A Python application that converts PDF text into speech so users can listen to documents like an audiobook.

## Features
- Select any PDF file
- Extracts text automatically
- Converts text to speech

## Technologies Used
- Python
- PyPDF2
- pyttsx3
- Tkinter

## How to Run
1. Install required libraries
pip install pyttsx3 PyPDF2

2. Run the program
python pdf_reader.py
